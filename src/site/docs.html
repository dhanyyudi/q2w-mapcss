A component kit for qgis2web exports.

q2w is a lightweight Tailwind + Leaflet UI framework. Drop it on top of an exported qgis2web map and get a polished header, footer, layer panel, popup, legend, and the rest — all theme-able, dark-mode ready, and built for production.

Tailwind · @apply Leaflet ~12kb gzip target light + dark + 4 presets a11y first
